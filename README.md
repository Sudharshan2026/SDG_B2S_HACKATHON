# Hometown Connect

A demo-ready civic-tech platform that helps people living outside their native district stay updated about their hometown. Provides hyperlocal information, public service updates, and complaint management.

## Features

- **Government Schemes** — Browse, search, filter, and bookmark schemes
- **Utilities Status** — Electricity and water schedules, outages, alerts
- **Local News** — Community and verified news, events, notices
- **AI Chatbot** — Q&A and complaint submission with ticket tracking
- **Authorities Directory** — Officials with contact info

## Demo Panels (No Login)

- **User** — View all content, submit complaints, track tickets
- **Admin** — Update utilities, manage assigned complaints
- **Super Admin** — Master data, all complaints, content moderation

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and switch between User, Admin, and Super Admin using the panel switcher.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- TypeScript
